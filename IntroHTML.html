Elements

Elements in HTML come in the form of:
Opening Tag
This describes what kind of element the following text or element will be.
It should always come before the text we're about to write.
Content
Content refers to...well the content of the element (aka the text).
Closing Tag
This tag is similar to the opening tag.
Instead of describing what kind of element the content is, it instead describes where the element ends.
